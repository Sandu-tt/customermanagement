# Customer Management System

##  Project Overview
This is a full-stack Customer Management System developed as part of a technical assignment.

The system allows managing customers, including:
- Personal details
- Mobile numbers
- Addresses
- Family members
- Bulk upload via Excel

---

##  Technologies Used

### Backend
- Java 
- Spring Boot
- Spring Data JPA (Hibernate)
- MariaDB
- Maven

### Frontend
- React.js
- Axios
- HTML / CSS

### Other
- Apache POI (Excel processing)
- Postman (API testing)

---

## 🚀 Features

### Backend Features
- Create Customer
- Update Customer
- Delete Customer
- Get All Customers
- Get Customer by ID
- Add Multiple Mobile Numbers
- Add Multiple Addresses
- Add Family Members
- Bulk Customer Upload using Excel

---

### Frontend Features
- Create Customer Form
- Update Customer (Edit)
- Delete Customer
- View Customer List
- Upload Excel File
- Responsive UI

---

## Excel Upload Format

Excel file must contain these columns:

| Name | DateOfBirth | NICNumber | MobileNumber |
|------|-------------|-----------|--------------|
